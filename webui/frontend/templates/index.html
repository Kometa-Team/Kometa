Libraries Configuration

View Documentation

Configure which Plex libraries Kometa should manage. Each library can have its own collection files, overlay files, metadata files, operations, and schedule. Library-level settings override global settings.

üí°
Library Configuration Tips:
‚Ä¢ Collection Files - Define what collections to create (actors, genres, charts, etc.)
‚Ä¢ Overlay Files - Add visual badges/overlays to posters (resolution, ratings, etc.)
‚Ä¢ Metadata Files - Manage metadata for individual items
‚Ä¢ Operations - Batch operations like mass genre updates, splits, or ratings sync
‚Ä¢ Use default: prefix for built-in Kometa files (e.g., default: basic)
