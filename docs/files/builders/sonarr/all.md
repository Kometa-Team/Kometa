# Sonarr All
    
Gets all Series in Sonarr.

???+ warning "Sonarr Configuration"

    [Configuring Sonarr](../../../config/sonarr.md) in the config is required for any of these builders.

### Example Sonarr All Builder(s)

```yaml
collections:
  ALL Sonarr Series:
    sonarr_all: true
```
