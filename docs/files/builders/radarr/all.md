# Radarr All

Gets all Movies in Radarr.

???+ warning "Radarr Configuration"

    [Configuring Radarr](../../../config/radarr.md) in the config is required for any of these builders.

### Example Radarr All Builder(s)

```yaml
collections:
  ALL Radarr Movies:
    radarr_all: true
```
